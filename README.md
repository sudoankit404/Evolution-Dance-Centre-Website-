# 🕺 Evolution Dance Centre — Official Website

> *Transform Your Passion Into Performance*

A bold, creative, and fully responsive website built for **Evolution Dance Centre (EDC)**,
a professional dance academy located in Uttam Nagar, New Delhi.
Affiliated with **Suro Bharti Sangeet Kala Kendra**.

---

## 🛠️ Built With

| Technology | Usage |
|---|---|
| **HTML5** | Page structure & semantic layout |
| **CSS3** | Styling, animations & responsive design |
| **JavaScript (Vanilla)** | Interactivity, scroll effects & form handling |
| **Font Awesome 6** | Icons throughout the website |
| **Google Fonts** | Poppins & Bebas Neue typography |

---

## ✨ Features

- 🎨 **Bold Creative Design** — Dark theme with vibrant gradient colors (Pink, Purple, Cyan, Gold)
- 💫 **Smooth Animations** — Scroll-triggered fade-ins, floating logo, parallax effects
- 📱 **Fully Responsive** — Mobile-first design, works on all screen sizes
- 🗺️ **Google Maps Integration** — Embedded location map
- 📬 **Contact Form** — Direct email integration via mailto
- 🎭 **Dance Forms Showcase** — Interactive cards for all 9 dance styles
- 🎬 **Visual Portfolio Gallery** — Linked to YouTube channel
- 📲 **Social Media Integration** — Instagram, Facebook, YouTube & Gmail links
- 🍔 **Mobile Navigation** — Hamburger menu for smaller screens
- ⏳ **Loading Screen** — Animated loader on page entry
- 🖼️ **Custom Logo** — Brand logo displayed in navbar section
